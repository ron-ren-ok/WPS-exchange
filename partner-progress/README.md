# 合作方数据汇报及达成预测

工作流每天北京时间 04:10 运行，读取 Google 表格「合作方新增血量」长表的最新回传数据，按合作方生成新增、血量及月目标预测。

- 初始默认发送至 `WPS_WEBHOOK_URL_TEST`。
- 手动触发时可选择 `test` 或 `production`。
- 生产切换时，将工作流中 `webhook_target` 的默认值改为 `production`，并将定时触发的 Webhook 表达式切换至 `WPS_WEBHOOK_URL`。
- 所有单位均以“万”展示；血量目标以“万美元”展示。
- 数据读取范围为长表 `A:E` 的完整已用区间，不设固定行数上限。
